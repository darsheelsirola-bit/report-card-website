# Grade Calculator

This is a simple Grade Calculator website made using **HTML, CSS, and JavaScript**.

## About the Project

The website takes a student's name and marks in three subjects:

* English
* Math
* Science

After entering the marks, the website calculates:

* Total marks
* Percentage
* Grade
* Final comment based on performance

## Features

* Simple and clean user interface
* Takes marks input from the user
* Checks if marks are between 0 and 100
* Calculates percentage automatically
* Shows grade and feedback
* Works directly in the browser

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Enter your name.
2. Enter marks for English, Math, and Science.
3. Click on the **Calculate Grade** button.
4. The report card will be shown on the screen.

## Grade Criteria

* A: 90% and above
* B: 80% and above
* C: 70% and above
* D: 50% and above
* F: Below 50% or failed in any subject

## Purpose

This project was made to practice basic web development concepts like forms, input fields, buttons, conditions, functions, and DOM manipulation in JavaScript.
